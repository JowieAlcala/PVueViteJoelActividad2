# PokemonVue

**PokemonVue** és un joc web inspirat en el món Pokémon, desenvolupat com a primer projecte de programació. Cada jugador pot escollir el seu equip Pokémon preferit d'una llista de 1025 criatures, per formar l'equip ideal.

## Descripció

Aquest projecte és un joc interactiu que permet als usuaris seleccionar el seu equip Pokémon de manera visual i intuïtiva. L'objectiu és oferir una experiència divertida i atractiva utilitzant tecnologies web modernes com HTML, CSS i JavaScript amb suport de Vue.js.

PokemonVue no només és un homenatge als clàssics jocs de Pokémon, sinó també una mostra d'aprenentatge i exploració personal en el món del desenvolupament web.

## Instal·lació

No cal instal·lació. El projecte s'executa directament al navegador. Només cal accedir a la pàgina web o obrir els fitxers HTML en local.

## Ús

1. Obre el fitxer `index.html` en el teu navegador.
2. Navega per la llista de Pokémon disponibles.
3. Selecciona els teus favorits per formar el teu equip ideal.
4. Diverteix-te creant combinacions!

## Característiques

- Llista completa de 1025 Pokémon disponibles per seleccionar.
- Interfície dinàmica i interactiva amb Vue.js.
- Disseny responsiu apte per a mòbils i ordinadors.
- Sense necessitat d’instal·lació ni backend.
